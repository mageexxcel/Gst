# Excellence_Gst
## Magento 2 Extension

### Excellence Gst easily manage your company's GST Information.



## Features:
1. Easily connect GST to your magento store.
2. Allow customers save their gst information.
3. The information is shown in invoice etc.
4. Easy to configure.
